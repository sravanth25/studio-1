# Build Project Workflow

This workflow ensures that all Node.js dependencies are installed via `npm install` and compiles the Next.js application for deployment readiness using `npm run build`.

1. **Install Dependencies**
// turbo
```bash
npm install
```

2. **Build the Application**
// turbo
```bash
npm run build
```
